# MTHS 112 — Business Mathematics Study App

Interactive study application for MTHS 112 (Business Mathematics). Features exam-style calculation problems with a workspace, custom math keypad, and memo-based marking.

## Topics Covered
- Matrices & Systems of Equations
- Functions, Revenue & Break-even
- Linear Programming
- Percentages & VAT
- Income Tax Calculations
- Financial Mathematics
- Calculus (Differentiation & Integration)

## Features
- **Workspace-based input** — Custom math keypad for entering calculations
- **Easy Mode** — Fill-in-the-blank guided templates
- **Hard Mode** — Blank workspace with self-assessment
- **Smart Marking Engine** — Step-by-step memo-based grading
- **Retry System** — Weak questions are flagged for guided retry

## Tech Stack
Vanilla HTML, CSS, and JavaScript with KaTeX for math rendering.
